# Связка глав 13-14

## Что проверяется

Проверяется переход:

```text
глава 13: контуры действия
-> глава 14: нейромедиаторы и гормоны как регуляторы режима контуров
```

Главный риск перехода: после аккуратной главы о контурах внезапно скатиться в популярную таблицу "вещество -> чувство".

## Критерии

| Критерий | Проверка | Статус |
| --- | --- | --- |
| Глава 14 продолжает главу 13 | В начале главы 14 прямо возвращается различение "контуры, а не центры", затем переносится на "медиаторы, а не состояния". | ок |
| Биохимия не подменяет контуры | Главная схема показывает медиаторы как регуляторы чувствительности, порогов, обучения, arousal, фильтрации, мобилизации и социальной значимости внутри контуров. | ок |
| Нет формулы "вещество -> чувство" | Глава прямо запрещает редукции: дофамин не равен мотивации, серотонин не равен настроению, окситоцин не равен любви. | ок |
| Есть учебный вход | Сначала вводятся нейромедиатор, нейромодулятор, гормон и рецептор; читателю не нужно заранее знать фармакологию. | ок |
| Есть нелинейность | Отдельно объяснены рабочий диапазон, inverted-U, tonic/phasic режимы, зависимость от задачи и состояния. | ок |
| Есть инженерный перевод | Для каждого блока есть вопросы к задаче, обратной связи, угрозе, шуму, управляемости, социальной рамке и восстановлению. | ок |
| Есть визуальная поддержка | В главе есть схема "медиаторы как регуляторы режима контуров" и таблицы перевода популярных формул в инженерные вопросы. | ок |
| Есть границы | Глава не дает медицинских, фармакологических, нутрицевтических или диагностических рекомендаций. | ок |
| Подготовлен переход к главе 15 | Финал главы переводит кортизол и стресс из "гормона стресса" в будущую тему мобилизации, аллостаза и окна полезной нагрузки. | ок |

## Что удержать при ревизии

- Не расширять главу в фармакологию и рецепторные подтипы.
- Не превращать таблицы в "вещество отвечает за".
- Не обещать практиками прямое управление медиаторами.
- Сохранить сквозной пример вечернего входа в туманную задачу: он связывает биохимию с уже введенными понятиями цены входа, обратной связи, управляемости, угрозы и внешнего контура мышления.
- При финальной редактуре можно усилить источниковую точность по новым работам 2024-2025, но не делать из них основу главы.
