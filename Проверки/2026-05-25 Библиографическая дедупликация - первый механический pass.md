# Библиографическая дедупликация - первый механический pass

Дата: `2026-05-25`.

Статус: `targeted-dedup-pass-1-done`.

## Область прохода

Проверен рабочий черновик единой публикационной библиографии:

`Источники/03-Единая-публикационная-библиография-рабочий-черновик.md`

Цель прохода - убрать явные повторы, которые уже можно надежно обнаружить механически:

- одинаковые DOI;
- одинаковые стабильные URL;
- одинаковые библиографические строки без учета служебной пометки в квадратных скобках.

Этот проход не закрывает полную финальную библиографию и не заменяет будущую сверку всех author-year ссылок reader-facing глав.

## Найденные повторы

Механическая проверка нашла 13 явных повторов:

- Tsutsui-Kimura et al. (2025) был продублирован между мотивационным разделом и нейрофизиологическим разделом.
- Блок организационных источников в разделе 7 повторял записи из раздела 5: JD-R, burnout, demand-control, effort-reward, recovery from work, job crafting, workload и ICD-11 burn-out.

## Принятые решения

- Tsutsui-Kimura et al. (2025) оставлен как полная запись в разделе 3, потому что это circuit-level источник по дофаминергическому механизму избегания в конфликте угрозы и награды. В разделе 2 оставлена короткая навигационная пометка без дублирования записи.
- Раздел 7 теперь оставляет только источники, которые добавляют организационный, лидерский или командный слой: SDT at work, psychological safety, Herzberg, work design, software-development interruptions, multidimensional work motivation.
- Базовые источники по выгоранию, восстановлению, JD-R, demand-control, effort-reward, job crafting и official burn-out classification оставлены в разделе 5.

## Проверки после правки

После правки выполнены проверки:

- duplicate pass по DOI - без совпадений;
- duplicate pass по стабильным URL - без совпадений;
- duplicate pass по точным библиографическим строкам - без совпадений;
- поиск trailing whitespace, search-псевдоссылок и служебных маркеров в рабочем файле библиографии - без совпадений.
