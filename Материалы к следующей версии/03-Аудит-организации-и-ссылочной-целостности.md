# Все ID, найденные во всех md-файлах корпуса, сверялись с ID реестра.

$stable = Join-Path $materialsRoot '02-Стабильные-якоря-текущей-версии.md'
# Все `Главы/*.md` из stable-карты проверялись через Test-Path от корня репозитория.

## Ограничения аудита

Этот аудит проверяет локальную организацию, локальные пути, ID материалов и устойчивость ссылок на главы. Он не является полной HTTP-проверкой внешних URL из research-пакетов и `safe_urls`. Внешние ссылки остаются библиографическими кандидатами и должны перепроверяться перед включением в текст книги.

## Правило для будущего агента

При использовании материалов не переносить номер главы механически. Сначала прочитать `02-Стабильные-якоря-текущей-версии.md`, определить `CV1-CHxx`, название главы и смысловой адрес, затем найти новое место в будущей структуре книги.
