## Что не удалось извлечь

Полные research-ответы не представлены в публичном backend JSON как обычные текстовые assistant-узлы: в цепочке есть технические заглушки `The output of this plugin was redacted.`. Поэтому основная evidence-база материала взята из пользовательских вложений, а не из shared JSON.

В файл `01-Извлеченная-беседа.md` намеренно не включены служебные элементы `system`, `tool`, скрытые custom-instruction placeholders и технические заглушки вместо tool/research-выводов.

Сырой список ссылок из `safe_urls` содержит технические, рекламные, поисковые и промежуточные элементы. Для книги его нельзя использовать как готовую библиографию без проверки по официальным архивам, издательским страницам, DOI, PubMed/PMC, библиотекам, музеям и документам организаций.
