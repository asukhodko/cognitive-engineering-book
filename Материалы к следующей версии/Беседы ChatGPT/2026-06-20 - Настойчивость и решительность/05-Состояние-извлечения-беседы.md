## Что не удалось извлечь

Полный research-ответ не представлен в публичном backend JSON как обычный текстовый assistant-узел: в цепочке есть технические заглушки `The output of this plugin was redacted.`. Поэтому основная evidence-база материала взята из пользовательского вложения, а не из shared JSON.

В файл `01-Извлеченная-беседа.md` намеренно не включены служебные элементы `system`, `tool`, скрытые custom-instruction placeholders, внутренние reasoning/thoughts-узлы, tool-вызовы в формате `code` и технические заглушки вместо tool/research-выводов.

Сырой список ссылок из `safe_urls` содержит дубли, технические и нерелевантные элементы. Для книги его нельзя использовать как готовую библиографию без проверки по DOI, PubMed/PMC, издательским страницам, университетским архивам и официальным документам.
