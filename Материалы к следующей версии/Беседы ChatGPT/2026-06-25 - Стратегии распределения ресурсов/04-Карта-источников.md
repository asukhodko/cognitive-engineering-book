# Карта источников

## Статус

Это рабочая карта evidence-блоков, практических систем, финансовых контуров и URL-кандидатов. Полный аннотированный корпус источников сохранен в `02-Research - стратегии распределения ресурсов.md`. Сырой список URL из backend JSON вынесен в `06-Ссылки-из-backend-json.md`; в этой беседе он почти не содержит полезной библиографии, потому что research-ответ в shared JSON скрыт.

## Основные блоки evidence

- **Task switching and attention residue**: switch costs, resumption lag, attention residue, цена фрагментации.
- **Fragmented knowledge work**: прерывания, meetings, reactive work, stress and protected focus time.
- **Cognitive load and external systems**: рабочая память, external offloading, reminders, GTD-like capture.
- **Sleep, recovery and energy**: сон, vigilance, executive functions, detachment, micro-breaks, movement.
- **Long working hours**: медицинские риски и пределы постоянного овертайма.
- **Goal systems**: goal-setting theory, implementation intentions, weekly commitments, 12-week cycles.
- **Productivity interfaces**: Deep Work, time blocking, Pomodoro, OKR, GTD как практические оболочки.
- **Developer productivity**: SPACE, DORA, protected focus time and ideal-vs-actual week.
- **Personal finance and resilience**: emergency fund, behavioral portfolio theory, budget layers, future goals not from leftovers.
- **Controversies and limits**: ego depletion, deliberate practice, универсальность Pomodoro/OKR/90-minute blocks.

## Очищенные URL-кандидаты из backend JSON

Содержательных URL-кандидатов в `safe_urls` после очистки нет; пользоваться библиографией из research-пакета.

## Проверка перед использованием в книге

- Заменить citation-маркеры вида `turn...` нормальными библиографическими ссылками.
- Отделить сильное evidence по переключениям, сну, восстановлению, целям и финансовому резерву от practitioner frameworks.
- Не подавать Pomodoro, OKR, GTD, Deep Work и 12 Week Year как научно доказанные универсальные законы; связывать их с более базовыми механизмами.
- Проверить первоисточники по WHO/ILO, AASM/SRS, CDC, OECD, CFPB, Microsoft Research, DORA and SPACE.
- В практической части явно развести масштабы: рабочий блок, день, неделя, квартал, год, жизненный портфель.
