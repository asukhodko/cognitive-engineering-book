## Что не удалось извлечь

Полные tool/research-выводы не представлены в публичном backend JSON как обычные текстовые assistant-узлы: в цепочке есть технические заглушки `The output of this plugin was redacted.`. Поэтому dossier нельзя считать полным research-пакетом.

В файл `01-Извлеченная-беседа.md` намеренно не включены служебные элементы `system`, `tool`, скрытые custom-instruction placeholders, внутренние reasoning/thoughts-узлы, tool/web/file-search вызовы в формате `code` и технические заглушки вместо tool/research-выводов.

Сырой список ссылок из `safe_urls` содержит технические, рекламные и промежуточные элементы. Для книги его нельзя использовать как готовую библиографию без проверки по DOI, PubMed/PMC, издательским страницам, университетским архивам и официальным документам.

## Команда извлечения

```powershell
curl.exe -x http://127.0.0.1:8899 -L --show-error --silent `
  --output "$env:TEMP\chatgpt-share-6a3f927a-93c4-83eb-a23a-ce2192e75fc2.json" `
  --write-out "%{http_code} %{content_type} %{size_download}" `
  -H "User-Agent: Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/125.0.0.0 Safari/537.36" `
  -H "Accept: application/json, text/plain, */*" `
  -H "Referer: https://chatgpt.com/share/6a3f927a-93c4-83eb-a23a-ce2192e75fc2" `
  "https://chatgpt.com/backend-api/share/6a3f927a-93c4-83eb-a23a-ce2192e75fc2"
```

Результат: `200 application/json`.

## Риск для будущего агента

Будущий агент может считать этот dossier рабочей копией видимой части shared-беседы и списка URL-кандидатов. Но он не должен считать, что публичная shared-ссылка содержит весь research-артефакт, и не должен принимать сырой `safe_urls` за готовую библиографию.
