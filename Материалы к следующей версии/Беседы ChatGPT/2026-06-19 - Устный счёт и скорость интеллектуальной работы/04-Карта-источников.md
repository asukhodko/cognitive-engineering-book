## Основные блоки evidence

- **Устный счёт и arithmetic fluency**: mental arithmetic, arithmetic fact retrieval, procedural arithmetic, mental abacus.
- **Базовые когнитивные механизмы**: working memory, attention control, processing speed, executive attention, speed-accuracy tradeoff.
- **Перенос тренировки**: near transfer, far transfer, active controls, working memory training, n-back, cognitive training.
- **Ускорение через обучение**: retrieval practice, deliberate practice, chunking, scaffolding, domain-specific automation.
- **Внешние опоры**: externalization, cognitive offloading, documentation, схемы, checklists, интерфейсное снижение cognitive load.
- **Физиологические ограничения**: сон, acute exercise, interruptions of sedentary time, micro-breaks, stress, burnout.
- **Профессиональный контекст**: job performance, multitasking work performance, software engineering cognitive load, code comprehension.

## Проверка перед использованием в книге

- Заменить citation-маркеры вида `turn...` нормальными библиографическими ссылками.
- Для каждого мета-анализа проверить DOI, журнал, N, inclusion criteria, active/passive controls и риск publication bias.
- Отделить быстрый счёт как частный навык от processing speed, attention control и working memory как более общих факторов.
- Не переносить выводы по детям и mental abacus напрямую на взрослых knowledge workers.
- Для практических рекомендаций отдельно указывать уровень доказательности: strong evidence, plausible but indirect, practitioner pattern.
