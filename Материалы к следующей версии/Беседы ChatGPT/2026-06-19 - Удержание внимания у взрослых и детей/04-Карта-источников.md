# Карта источников

## Статус

Это рабочая карта источниковых блоков. Полные формулировки, citation-маркеры, ссылки и дополнительные оговорки сохранены в `02-Research - удержание внимания у взрослых и детей.md`. Сырой список URL из backend JSON вынесен в `06-Ссылки-из-backend-json.md` и не должен использоваться как готовая библиография.

## Источниковые блоки

### Рабочая память и task set

Нужны источники о working memory capacity, executive attention, task set maintenance, goal maintenance и control of action.

### Сети внимания и default mode

Нужны источники о frontoparietal control network, dorsal attention network, ventral attention network, salience network, default mode network и mind wandering.

### Цена переключений и цифровые прерывания

Нужны источники о task switching cost, attention residue, interruptions, notifications, media multitasking и восстановлении после прерывания.

### Недосып, усталость и когнитивный контроль

Нужны источники о sleep deprivation, vigilance, sustained attention, mental fatigue, PFC и executive control under fatigue.

### Дети и развитие executive functions

Нужны источники о возрастном развитии working memory, inhibitory control, cognitive flexibility, self-regulation и внешних scaffold-практиках для детей.

### ADHD и дифференциальная диагностика

Нужны источники о ADHD, executive dysfunction, sustained attention, clinical assessment, behavioral interventions и ограничениях самопомощи.

### Практические интервенции и организационный дизайн

Нужны источники о implementation intentions, environmental design, Pomodoro/focus blocks, external memory aids, interruption management, classroom/workplace supports и измерении прогресса.

## Проверка перед использованием в книге

- Разделить evidence для взрослых, детей и ADHD-клинических групп.
- Проверить DOI/PMID/официальные страницы источников из research-файла.
- Развести лабораторные эффекты task switching и реальные рабочие дни.
- Отдельно отметить, где данные относятся к цифровым прерываниям, а где к более широкому multitasking.
- Не использовать citation-маркеры `turn...` как библиографию.
