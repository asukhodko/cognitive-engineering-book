## Ограничения

Полные tool/research-выводы после команды `@Глубокое исследование` не представлены в публичном backend JSON как обычные текстовые assistant-узлы: в цепочке есть технические заглушки `The output of this plugin was redacted.`. Поэтому основная evidence-база материала взята из пользовательского вложения, а не из shared JSON.

В файл `01-Извлеченная-беседа.md` намеренно не включены служебные элементы `system`, `tool`, скрытые placeholders, внутренние `thoughts`/`reasoning_recap`, tool/web/file-search вызовы в формате `code` и технические redacted-заглушки.

Сырой список ссылок из `safe_urls` содержит технические, рекламные и промежуточные элементы. Для книги его нельзя использовать как готовую библиографию без проверки по DOI, PubMed/PMC, издательским страницам, университетским архивам и официальным документам.
