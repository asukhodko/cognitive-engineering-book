# Материалы к следующей версии

Этот каталог - входящий корпус для будущей версии книги. Основной рабочий вход: `00-Навигатор.md`.

Короткий порядок чтения:

1. `00-Навигатор.md` - назначение каталога, правила добавления материалов и порядок работы будущего агента.
2. `01-Реестр-материалов.md` - полный список материалов и редакторских требований.
3. `02-Стабильные-якоря-текущей-версии.md` - карта глав текущей версии, к которой относятся номера глав в материалах.
4. `03-Аудит-организации-и-ссылочной-целостности.md` - результат проверки локальных путей, ID материалов и stable-якорей.

Важно: номера глав внутри материалов относятся к snapshot `CV1` первой публичной/current reader edition. В реестре они продублированы stable ID вида `CV1-CH07 (гл. 7)`. При подготовке новой версии книги эти номера нужно использовать как исторические указатели, а не как гарантированные номера будущих глав.
