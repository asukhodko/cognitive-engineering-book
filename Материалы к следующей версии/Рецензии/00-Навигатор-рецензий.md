# Рецензии

## Назначение

Этот каталог хранит рецензии на текущую версию книги и связанные рабочие выжимки для следующего редакторского прохода.

Каждую крупную рецензию лучше сохранять отдельным каталогом:

```text
YYYY-MM-DD - Автор или источник - краткое название/
  00-Карточка-рецензии.md
  01-Полный-текст-рецензии.md
  02-Выжимка-и-решения.md
```

## Как использовать

- `01-Полный-текст-рецензии.md` сохраняет исходную рецензию без содержательной переработки.
- `00-Карточка-рецензии.md` фиксирует источник, статус, приоритет и основные выводы.
- `02-Выжимка-и-решения.md` переводит рецензию в рабочие редакторские задачи.

После добавления рецензии нужно занести ее в `../01-Реестр-материалов.md`.
